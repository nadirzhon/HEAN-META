## Objective
Проанализировать файл `$ARGUMENTS[0]`, найти в нем "узкое место" (например, сложный цикл или математические вычисления) и сгенерировать "скелет" реализации этой функции на Rust, используя `rust_bridge.py`.

## Warning
Это очень сложный и экспериментальный скилл. Результат потребует ручной доработки.

## Workflow
1.  Прочитай и проанализируй указанный Python-файл.
2.  Найди функцию, которая является хорошим кандидатом для оптимизации.
3.  Изучи `src/hean/physics/rust_bridge.py`, чтобы понять, как работает взаимодействие Python и Rust в этом проекте.
4.  Создай новый `.rs` файл с шаблонной реализацией выбранной функции на Rust.
