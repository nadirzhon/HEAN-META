## Objective
Подготовить лучшую стратегию из последнего запуска `/run-evolution-cycle` к тестированию в реальной среде.

## Workflow
1.  Проанализируй вывод последнего запуска `/run-evolution-cycle`, чтобы найти геном с наивысшим Sharpe Ratio.
2.  Создай фрагмент конфигурации для файла `.env`, который активирует эту стратегию (если для нее есть отдельный флаг) или изменяет параметры существующей стратегии.
3.  Покажи этот фрагмент пользователю для одобрения.
