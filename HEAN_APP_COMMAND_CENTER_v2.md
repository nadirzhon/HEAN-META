# HEAN iPHONE APP — КОМАНДНЫЙ ЦЕНТР v2.0

## Философия: Не Дашборд, а Боевой Штаб

> **Старый подход:** Приложение показывает цифры. Ты смотришь и не понимаешь.
> **Новый подход:** Приложение — это AI-наставник который ОБЪЯСНЯЕТ каждое движение рынка, 
> ПОКАЗЫВАЕТ визуально что происходит, СПРАШИВАЕТ разрешение перед действием, 
> и УЧИТ тебя понимать рынок.

### Три Принципа

1. **ПРОЗРАЧНОСТЬ** — ни одно решение AI не скрыто. Ты видишь ВСЮ логику.
2. **КОНТРОЛЬ** — AI предлагает, ТЫ решаешь. Без твоего подтверждения ни одна сделка не откроется.
3. **ОБУЧЕНИЕ** — каждый элемент экрана можно нажать и получить объяснение "что это и зачем".

## Ключевой элемент: ОБЪЯСНИТЕЛЬНАЯ КАРТОЧКА

Это серый блок с 💬 наверху. AI каждые 30 секунд обновляет текст и **простым языком** объясняет что происходит.

### Как AI формирует объяснение

В промпте Claude добавляется специальная инструкция:

```
ДОПОЛНИТЕЛЬНО К АНАЛИЗУ:
Напиши объяснение для человека который УЧИТСЯ торговать.
Правила:
- Максимум 4 предложения
- Простой язык, без жаргона
- Объясни ПОЧЕМУ рынок ведёт себя так
- Дай одну конкретную рекомендацию (ждать/готовиться/быть осторожным)
- Используй аналогии если помогает понять

Формат:
"explanation": "текст объяснения для пользователя"
```

```swift
    @State private var showExplanation = false
    
    var body: some View {
        // Нажимаемая карточка
        // При нажатии → showExplanation = true
        // .sheet(isPresented: $showExplanation) { ExplanationSheet(data: explanation) }
    }
}

struct ExplanationData {
    let title: String
    let whatIsIt: String           // Что это
    let howToRead: String          // Как читать

    let formula: String?           // Формула (опционально)
    let chartData: [Double]?       // Данные для мини-графика
}
```

## Ключевая фишка: "Для тебя"

Каждый анализ содержит блок **"Для тебя"** — это объяснение аналогиями, простым языком, как если бы опытный трейдер сидел рядом и объяснял новичку.

В промпте Claude:
```
БЛОК "ДЛЯ ТЕБЯ":
Напиши объяснение как для друга который хочет понять трейдинг.
- Используй бытовые аналогии (магазин, погода, спорт)
- Объясни ПОЧЕМУ ты видишь то что видишь
- Дай один совет — ждать, готовиться, или быть осторожным
- Максимум 3-4 предложения
```

```swift
// Глобальный modifier для всех view
struct ExplainableModifier: ViewModifier {
    let explanation: String
    @State private var showPopover = false
    
    func body(content: Content) -> some View {
        content
            .onLongPressGesture {
                showPopover = true
            }
            .popover(isPresented: $showPopover) {
                Text(explanation)
                    .padding()
                    .presentationCompactAdaptation(.popover)
            }
    }
}
```

## Объяснения — откуда брать

Все объяснения генерируются ДВУМЯ способами:

```swift
struct ExplanationDatabase {
```
