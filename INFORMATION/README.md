## Risk Management

### Position Sizing
- Max risk per trade: **1% of capital**
- Max position in single asset: **20% of capital**
- Max concurrent positions: **5-10**

### Stop-Losses
- Funding arb: Exit if funding < **0.02%**
- Liquidation hunting: **1-2% stop-loss**
- Sentiment trading: **2-3% stop-loss**
- Cross-exchange arb: **0.5% stop-loss**

### Killswitch
- Drawdown: **>20% from peak** ‚Üí halt all trading
- Consecutive losses: **5 in a row** ‚Üí pause and review
- System errors: Repeated API failures ‚Üí backup exchange
