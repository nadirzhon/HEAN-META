
## 💡 КЛЮЧЕВЫЕ КОНЦЕПТЫ

### Оптимизация:
- `React.memo` - предотвращает лишние рендеры
- `useMemo` - кэширует дорогие вычисления
- `useCallback` - стабильные ссылки на функции
- `Debouncing` - отложенное выполнение

### Анимации:
- `Framer Motion` - библиотека анимаций
- `AnimatePresence` - анимация входа/выхода
- `layout` prop - smooth layout transitions
- `whileHover` - hover эффекты
