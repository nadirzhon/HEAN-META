## ⏱️ TIMELINE ВСЕХ АНИМАЦИЙ

```
Время    | Событие
---------|--------------------------------------------------
0ms      | Новое событие пришло
0-50ms   | Счётчик увеличивается (scale 1.3)
50ms     | Событие начинает появляться (opacity 0 → 0.3)
100ms    | Цветная полоса начинает расти
150ms    | Событие полностью видимо (opacity 1, y: 0)
200ms    | Spring bounce завершён
250ms    | Payload начинает раскрываться
300ms    | Первое поле payload видимо
350ms    | Второе поле payload видимо
400ms    | Третье поле payload видимо
450ms    | Четвёртое поле payload видимо
500ms    | Все анимации завершены

+ Параллельно:
- Pulse эффект для ERROR (каждую 1s, 3 раза)
- Activity индикатор пульсирует (каждые 2s, бесконечно)
- Spinner поиска крутится (пока debounce)
```

## 🎬 СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### Сценарий 1: Мониторинг торговли

```
1. Пользователь открывает Event Feed
   → Видит пульсирующий 💚 = система живая

2. Приходит ORDER_FILLED событие
   → fade-in + slide down
   → Голубая полоса слева
   → Появляется сверху
   → Пользователь сразу замечает

3. Приходит ERROR событие
   → fade-in + slide down
   → Красная полоса слева
   → 🔴 PULSE эффект 3 раза
   → Невозможно не заметить!
```

### Сценарий 2: Поиск проблемы

```
1. Пользователь вводит "error" в поиск
   → ⚪ Spinner показывает что идёт поиск

2. Через 300ms результаты появляются
   → События плавно фильтруются
   → Spinner исчезает

3. Наводит мышку на событие
   → Карточка увеличивается
   → Появляется тень
   → Легко читать
```

### Сценарий 3: Фильтрация

```
1. Пользователь кликает на "ORDERS"
   → Кнопка сжимается (scale 0.95)
   → Feedback мгновенный

2. События фильтруются
   → Ненужные плавно исчезают
   → Нужные остаются
   → Layout плавно адаптируется
```

## 💡 ПОЧЕМУ ЭТО ВАЖНО

### Без анимаций:
```
❌ События появляются резко
❌ Непонятно что изменилось
❌ ERROR события теряются
❌ Нет feedback на действия
```

### С анимациями:
```
✅ Плавное появление событий
✅ Сразу видно новые изменения
✅ ERROR события выделяются
✅ Каждое действие подтверждено
```
