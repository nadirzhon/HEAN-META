## ⚡ Быстрая диагностика проблем

### Проблема: "Docker daemon is not running"
**Решение:**
```bash
open -a "Docker Desktop"
# Подождите 30 секунд
./start_hean_complete.sh
```

### Проблема: "Port 8000 already in use"
**Решение:**
```bash
lsof -ti:8000 | xargs kill -9
./start_hean_complete.sh
```

### Проблема: "Connection to Bybit failed"
**Решение:**
1. Проверьте интернет
2. Проверьте API ключи в `backend.env`
3. Убедитесь что используете **testnet** ключи

### Проблема: "Container keeps restarting"
**Решение:**
```bash
# Посмотрите логи
docker compose logs symbiont-testnet | tail -50

# Если нужно - пересоберите образ
docker compose build --no-cache symbiont-testnet
docker compose up -d symbiont-testnet
```
